# Webapp Builder Platform

Welcome to the **Webapp Builder Platform**! This platform provides a suite of tools and templates for rapidly building and deploying modern web applications.

## Features

- **Modern Architecture**: Built with scalable and performant technologies.
- **Rich Templates**: A collection of beautiful, ready-to-use templates across various industries.
- **Customizable**: Easy to tweak and customize according to your needs.
- **Responsive Design**: Mobile-first and fully responsive out of the box.

## Getting Started

1. **Clone the repository:**
   ```bash
   git clone https://github.com/gk-girishkumar/Webapp_Builder_Platform.git
   ```
2. **Install dependencies:**
   Navigate into the project directory and run:
   ```bash
   npm install
   ```
3. **Run the development server:**
   ```bash
   npm run dev
   ```
